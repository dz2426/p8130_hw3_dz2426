# Description of “Instacart” dataset

Comments:

write a short description of the dataset, noting the size and structure
of the data, describing some key variables, and giving illstrative
examples of observations

This limited “Instacart” dataset contains 1,384,617 observations of
131,209 unique users, and 15 columns of variables. Some key variables
are “aisle” “department”, “product\_name”, “reordered” showing whether
the item has been ordered by this user in the past, and “order\_dow”
indicating the day of the week on which the order is placed.

“Aisle” has 134 groups, such as yogurt, fresh vegetables, and fresh
fruits. “Product\_name” includes 39123 products, such as Bulgarian
Yogurt, Organic Celery Hearts, and Bag of Organic Bananas. “Department”
includes 21 departments, such as dairy eggs, produce, canned goods.
